# What Happens When You Type https://www.google.com in Your Browser and Press Enter?

## Description

This project explores what happens behind the scenes when you type a URL into your browser and press Enter. It covers the full journey of a web request, from DNS resolution to the database, and back to the browser rendering the page.

## Files

| File | Description |
|---|---|
| `0-blog_post` | URL to the published blog post on LinkedIn |
| `1-what_happen_when_diagram` | URL to the diagram image illustrating the request flow |
| `blog_article.md` | Full article explaining each step of the process |

## Concepts Covered

- DNS resolution
- TCP/IP three-way handshake
- Firewall filtering
- HTTPS/TLS encryption
- Load balancing
- Web server
- Application server
- Database
